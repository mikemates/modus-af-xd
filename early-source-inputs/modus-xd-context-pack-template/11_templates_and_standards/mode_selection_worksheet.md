## Mode Guidance

| Project condition | Default mode |
|---|---|
| Ambiguous opportunity; solution not clear | Mode 0 |
| Greenfield utility app; minimal brand; narrow users | Mode 1 |
| Greenfield B2B enterprise tool; moderate brand/system needs | Mode 1 or 2 |
| Customer-facing MVP; brand matters but speed matters more | Mode 2 |
| B2C product with broad audience and high trust needs | Mode 2 or 3 |
| Brownfield product inside existing ecosystem | Mode 3 |
| Mature client design system with approval expectations | Mode 3 |
| Regulated, accessibility-sensitive, or complex workflow environment | Mode 3 |
